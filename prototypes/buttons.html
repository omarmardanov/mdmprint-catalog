Кнопки

Форма — пилюля, едина для всего сайта. Четыре вида, три размера. Всё на токенах из tokens.css, стили — kit.css, класс .mk-btn.

Виды

Вид выбирается по весу действия, не по вкусу. На экране — одна primary.

Primaryглавное действие
Darkдействие интерфейса
Ghostвторичное действие
Linkтретьестепенное
Где применяется. Primary — CTA в hero, финальный CTA, отправка формы, «Оставить заявку» в топбаре. Dark — «Каталог», «Найти» в поиске. Ghost — «Заказать образец», «Калькулятор», второй CTA рядом с primary. Link — «Заказать звонок» и всё, что не должно перетягивать внимание.

Размеры

Три штуки. Размер задаёт контейнер, а не важность действия.

lg — 48pxпо умолчанию
md — 36px.mk-btn--md
sm — 32px.mk-btn--sm
Только иконка.mk-btn--icon
Правило. lg — контент страницы. md — панель шапки (высота 36 при полосе 54). sm — топбар (высота 32 при полосе 40) и кнопка внутри поля поиска.

Состояния

Наведите мышь — живой hover. Ниже те же состояния зафиксированы классами .is-hover / .is-active, чтобы их было видно на скриншоте.

Primary#FFC100 → #E6A800 → #CC9E00
Dark#222220 → #000 → #3A3A38
Ghostрамка → чёрная, фон → #F6F6F4
Linkпунктир → сплошная
Focusтолько с клавиатуры
Focus-ring — 2px --brand-dark с отступом 2px, показывается только при навигации с клавиатуры (:focus-visible), мышь его не вызывает. Disabled — плоский серый --surface-gray для всех видов (не прозрачность: полупрозрачный жёлтый читается как активная кнопка). Клики не проходят.

На тёмном фоне

Финальный CTA-блок и футер. Dark-кнопка здесь не работает — только primary и ghost-инверсия.

Ghost на тёмном пока задан инлайном. Если блок «финальный CTA» останется тёмным — завести .mk-btn--ghost-inverse в ките.

Что не сходится с прототипом

Зафиксировано, не исправлено. Требует решения.

1. Жёлтый. Шапка красит CTA в #FFC829, бренд-токен — #FFC100. Кит использует токен.
2. Радиус. usluga.html:26 объявляет локальный --radius-pill:40px и затирает токен 999px. На высоте 48 разница не видна, на 32 — видна.
3. Высоты 28 и 40. Встречаются по одному разу (.usl .btn-sm, кнопка в модалке). В системе их нет — при миграции схлопнуть в 32/36.
4. Кнопки прототипа (.btn, .catbtn, .calc, .s-go, .tb-cta) пока живут своим CSS. Миграция на .mk-btn — отдельный шаг.
