# Отработавшие песочницы

**К передаче в разработку не относятся. Читать не нужно.**

Здесь лежат черновые страницы, на которых отдельные блоки собирались и сравнивались в вариантах,
прежде чем попасть в `prototypes/usluga.html`. У большинства содержимое **уже перенесено в боевой
прототип** — это история решений, а не исходники (исключения — три нижних, см. под таблицей).

| Файл | Что решалось |
|---|---|
| `_sandbox-adv.html` | Преимущества на мобилке |
| `_sandbox-btn2.html` | Вторичная кнопка |
| `_sandbox-catalog.html` | Каталог-рельса на мобилке |
| `_sandbox-faq.html` | Аккордеоны ЧАВО |
| `_sandbox-pane.html` | Экран раздела каталога на мобилке |
| `_sandbox-pm.html` | Блок «Что учесть» на десктопе |
| `_sandbox-price.html` | Строки цен на мобилке |
| `_sandbox-prod.html` | «Где применяется» на мобилке |
| `_sandbox-req.html` | Требования к макету |
| `_sandbox-specs.html` | Блок «Характеристики» (шаблон материалов) — **не согласован, не перенесён** |
| `_sandbox-form.html` | Единый компонент формы (D-паттерн + М2) — эксперимент |
| `_sandbox-inputs.html` | Инпуты для форм и калькуляторов — эксперимент |

Не удалены намеренно: в них видно, какие варианты отвергнуты и почему. Если по блоку возникнет
спор «а почему сделано так», ответ обычно здесь.

**Три последних — не «отработавшие», а отложенные.** `_sandbox-specs` собран, но ещё не согласован
и в `usluga.html` не перенесён (см. `DEV_WALKTHROUGH.md` §4.1). `_sandbox-form` и `_sandbox-inputs` —
черновые эксперименты к «свести три формы в один компонент» (отложено до WordPress). Перенесены сюда
из `prototypes/`, чтобы витрина прототипов оставалась только эталонами.
