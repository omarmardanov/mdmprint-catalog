# Отработавшие песочницы

**К передаче в разработку не относятся. Читать не нужно.**

Здесь лежат черновые страницы, на которых отдельные блоки собирались и сравнивались в вариантах,
прежде чем попасть в `prototypes/usluga.html`. Их содержимое **уже перенесено в боевой прототип** —
это история решений, а не исходники.

| Файл | Что решалось |
|---|---|
| `_sandbox-adv.html` | Преимущества на мобилке |
| `_sandbox-btn2.html` | Вторичная кнопка |
| `_sandbox-catalog.html` | Каталог-рельса на мобилке |
| `_sandbox-faq.html` | Аккордеоны ЧАВО |
| `_sandbox-pane.html` | Экран раздела каталога на мобилке |
| `_sandbox-pm.html` | Блок «Что учесть» на десктопе |
| `_sandbox-price.html` | Строки цен на мобилке |
| `_sandbox-prod.html` | «Где применяется» на мобилке |
| `_sandbox-req.html` | Требования к макету |

Не удалены намеренно: в них видно, какие варианты отвергнуты и почему. Если по блоку возникнет
спор «а почему сделано так», ответ обычно здесь.

**Единственная песочница, которая НЕ здесь** — `prototypes/_sandbox-specs.html` (блок
«Характеристики» для шаблона материалов). Она не отработала: блок собран, но ещё не согласован
и в `usluga.html` не перенесён.
